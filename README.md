# Quote Generator Web Application

A simple Quote Generator web application developed using Python, HTML, and database integration during Skill Development classes.

## About the Project
This application allows users to add and generate quotes along with author name and domain/category.  
After entering the details and clicking generate, the application displays the formatted quote.

### Example Output
> "Where there is a will there is a way."  
> — Deepika | Work

## Features
- Add custom quotes
- Add author name
- Add domain/category
- Generate and display quotes dynamically
- Simple and user-friendly interface

## Technologies Used
- Python
- HTML
- Database

## Project Structure
- `main.py` → Main application logic
- `database.py` → Database connection and operations
- `index.html` → Frontend interface

## How to Run
```bash
python main.py
```
