# ✨ Quote Generator Web Application

A simple and interactive Quote Generator Web Application developed during Skill Development classes using Python, HTML, and database integration.

---

## 📌 About the Project

This application allows users to add and generate quotes along with the author name and domain/category.  
After entering the details and clicking the generate button, the application displays the formatted quote dynamically.

### 📝 Example Output

> “Where there is a will there is a way.”  
> — Deepika | Work

---

## 🚀 Features

- Add custom quotes
- Add author names
- Add domain/category
- Generate quotes dynamically
- Simple and user-friendly interface
- Database integration for storing quotes

---

## 🛠️ Technologies Used

- Python
- HTML
- SQLite Database
